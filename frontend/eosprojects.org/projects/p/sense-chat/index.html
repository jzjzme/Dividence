Description

Decentralized messaging built on EOS - Connect directly, chat privately, transact freely. Sense Chat is a full featured messenger with a built in EOS wallet. Users can make Live Chat text conversations, video calls, and transact EOS tokens and other digital assets. Sense Chat creates direct connections between users and does not use any centralized storage.
